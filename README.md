# Дерево математических выражений

Программа строит бинарное дерево выражений по введенной математической формуле, умеет вычислять её значение и автоматически упрощать формулу, вынося общий множитель за скобки.

## Что делает программа

1. Строит дерево из обычной текстовой строки (инфиксной записи) с учетом приоритета операций `+`, `-`, `*`, `/`, унарных минусов и скобок (макс. глубина вложенности — 10).
2. Упрощает формулу по правилам вынесения общего множителя:
   * (f1 * f3) + (f2 * f3) -> (f1 + f2) * f3
   * (f1 * f2) + (f1 * f3) -> f1 * (f2 + f3)
3. Вычисляет результат, запрашивая у пользователя числовые значения для буквенных переменных.

## Данные для проверки

| Входная формула | Результат упрощения | Значения переменных | Результат вычислений |

| `((a*c)+(b*c))` | `((a+b)*c)` | `a=2, b=3, c=4` | `20` |

| `(((5*8)-(9*5))+(8/4))` | `((5*(8-9))+(8/4))` |  | `-3` |

| `(x*y-x*z)` | `(x*(y-z))` | `x=10, y=5, z=3` | `20` |
